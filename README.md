# AIギャルゲーリポジトリ

これはAIギャルゲーのリポジトリです。

## 概要

このリポジトリは、AIを使用したギャルゲー（恋愛シミュレーションゲーム）のソースコードを含んでいます。

## 使い方

1. リポジトリをクローンします。
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```

2. 必要な依存関係をインストールします。
    ```bash
    pip install -r requirements.txt
    ```

3. ゲームを開始します。
    ```bash
    python main.py
    ```
